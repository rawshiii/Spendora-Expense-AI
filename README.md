# Spendora - ExpenseAI 💸

Spendora is a full-stack personal finance management application built using the MERN Stack. It helps users track expenses, manage budgets, set financial goals, and analyze spending habits through interactive dashboards and AI-powered insights.

## Features

* User Authentication (JWT)
* Expense Tracking
* Add, Edit and Delete Expenses
* Budget Management
* Financial Goal Tracking
* Analytics Dashboard
* AI Financial Insights
* Receipt Upload
* PDF Export
* CSV Export
* Calendar View
* User Profile

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios
* Recharts
* React Hook Form

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer
